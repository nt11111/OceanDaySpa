# Ocean Day Spa — Website

A clean, ocean-branded single-page site for **Ocean Day Spa** (1989 Santa Rita Rd Ste B, Pleasanton, CA).
React + TypeScript, compiled in the browser — **opens by double-clicking `index.html`** (no install, no server).

## Brand & design
- **White background, ocean-blue accents** (brand blue `#1962A6`, sampled from the real logo).
- Real **logo** (`images/logo.png`), badge (`images/badge.png`, also the favicon) and brand collages (`hf_*` images) used throughout.
- Animated, interactive: scroll reveals, count-up stats, animated wave divider, floating brand chips, hover zoom, marquee, shimmer buttons, gallery lightbox, sticky mobile call bar.
